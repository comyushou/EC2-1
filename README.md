### NAGANO-CAKE
===================

長野県にある小さな洋菓子店「ながのCAKE」の商品を通販するためのECサイト開発。
元々近隣住⺠が顧客だったが、昨年始めたInstagramから⼈気となり、全国から注⽂が来るようになった。元々近隣住⺠が顧客だったが、昨年始めたInstagramから⼈気となり、全国から注⽂が来るようになり、管理機能を含む通販サイトを作成することとなった。

### 使い方
===================

_インストール方法_
   $ git clone
   $ bundle install
   $ rails db:migrate
   $ rails db:seed
   $ rails s -b 0.0.0.0
